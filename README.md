## Key-map 
- ,w: save
- ctrl + h/j/k/l: switch window
- ,t: create tag file
- ,c: clean up all the space at the end of the line
- F3: Open or close NERDTree
- F4: open or close Tagbar
- F5: find C symbols  
- F6: find string
- F7: find function called
